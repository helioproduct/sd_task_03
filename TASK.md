# Домашнее задание 03: Проектирование и оптимизация реляционной БД

## Вариант 9. Электронная почта

Сущности:

- пользователь
- почтовая папка
- сообщение

Результат работы оформлен в виде набора SQL- и Markdown-артефактов:

- `schema.sql` - создание схемы БД, ограничений и индексов;
- `data.sql` - тестовые данные;
- `queries.sql` - SQL-запросы для операций варианта;
- `optimization.md` - описание оптимизаций и планы выполнения `EXPLAIN`;
- `README.md` - описание схемы БД и инструкции по запуску.

Для локального запуска PostgreSQL используется `docker-compose.yaml`.
